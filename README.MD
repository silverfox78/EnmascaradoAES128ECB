# Texto Ofuscado

## Requerimientos

* Para el Token
    + Llave de firmado
    + Tiempo de expiracion


* Para la ofuscacion, se requiere una llave de comun acuerdo, e idealmente algo que saltee la informacion, como la fecha por ejemplo.

-----

* crypto: "^1.0.1",
* jsonwebtoken: "^8.5.1"
